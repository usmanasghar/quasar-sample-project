# Shipvagoo Front-end

## Getting Started

### Prerequisites

- Latest stable Node runtime (v18 at the time of writing)
- yarn package manager

### Installation

1. Clone the project



2. cd to the cloned project
3. Install the dependencies by executing yarn in the project root

```sh
yarn
```

4. Set-up husky hooks
